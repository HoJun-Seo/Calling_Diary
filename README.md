# Calling Diary

## 프로젝트 설명
특정 날짜에 작성해둔 일정이 있을 때, 사용자가 정해둔 시간에 맞춰 해당 일정에 대한 알림 문자를 발송해주는 일정 알림 웹 사이트 입니다.
- 원하는 날짜를 지정하여 시간 별로 원하는 일정을 작성할 수 있습니다.
- 각 일정 별로 휴대폰 문자 메시지를 통해 원하는 시간대에 알림을 받을지 말지 결정할 수 있습니다.
